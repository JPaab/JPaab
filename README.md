## · What I build

I learn backend by building projects with real product flows: APIs with authentication, database persistence, documentation, tests and user-facing features around them.

* REST APIs with authentication, authorization, validation, error handling and OpenAPI documentation.
* Backend applications structured with controllers, services, repositories, DTOs and mappers.
* SQL-based persistence with JPA/Hibernate, MySQL and PostgreSQL.
* Tests with JUnit 5, Mockito and Spring Boot Test.
* Full-stack flows: catalog, product pages, cart, checkout, orders and payment integration.
* Interfaces with attention to detail when the product needs more than just backend logic.

## · Qué desarrollo

Estoy aprendiendo backend convirtiendo proyectos prácticos en aplicaciones con forma de producto: APIs con autenticación, persistencia en base de datos, documentación, tests y flujos reales.

* APIs REST con autenticación, autorización, validación, manejo de errores y documentación OpenAPI.
* Aplicaciones backend estructuradas con controllers, services, repositories, DTOs y mappers.
* Persistencia SQL con JPA/Hibernate, MySQL y PostgreSQL.
* Tests con JUnit 5, Mockito y Spring Boot Test.
* Flujos full-stack: catálogo, fichas de producto, carrito, checkout, pedidos e integración de pagos.
* Interfaces cuidadas cuando el producto necesita algo más que lógica backend.
